MangleBracket is an interactive Word to HTML conversion tool.

UPDATE: Word conversion ran on a separate server I no longer felt like paying for. MangleBracket continues to operate, but only OpenOffice documents. Convert to OpenOffice ODT before continuing.

Remember when Aperture talked about integrating into your digital RAW workflow? Well MangleBracket integrates into your web copy workflow — a workflow where you receive a Word document from the copywriter (or an external client) and convert it to clean generic HTML which can pick up styles from your CSS stylesheets.

Never again have to deal with:

  • Wrapping text in P and LI tags
  • Character encoding issues for symbols™, forëign characters, and “smart” quotes
  • Searching for placeholder hrefs
  • Hand converting a 20 page Word doc only to learn that the copywriter has an updated Word doc with changes scattered all over the place
  • Word's native "Export to HTML"
